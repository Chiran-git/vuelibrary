## Library

A Library Crud created in Laravel and Vue 


## Installation steps

```
git clone https://github.com/Chiran-git/commercepundit.git
```

```
composer install
```

Create .env

Add database details

```
php artisan key:generate
```

```
npm install
```

```
php artisan migrate
```

```
php artisan db:seed
```
